---

Generate beautiful, random confetti patterns on your canvas with customizable shapes, colors, density, and size. Perfect for adding celebratory flair to your designs, presentations, and collaborative sessions.

Works Across All Figma Apps

Confetti Genie is fully compatible with:

- Figma Design - Add confetti to your design frames and components
- FigmaJam - Spice up your brainstorming sessions and whiteboards
- Figma Slides - Create engaging presentation backgrounds
- Figma Buzz - Enhance your interactive experiences

Features

🎨 Shape Variety

Choose from multiple confetti shapes:

- Rectangles - Classic rectangular pieces
- Ellipses - Circular confetti elements
- Polygons - Multi-sided geometric shapes (3-6 sides)
- Stars - Sparkling star shapes (4-6 points)

You can select one or more shape types, or use all of them together for maximum variety. All shapes are enabled by default.

🎨 Color Control

- Custom Colors - Add your own color palette using the color picker. Build a collection of colors that match your brand or design theme
- Random Colors - Let Confetti Genie generate vibrant, random colors for a spontaneous celebration effect
- Color Palette Management - Add multiple colors, remove individual colors, or reset to default

⚙️ Customization Options

- Density Control - Adjust the number of confetti pieces (30-300) with an intuitive slider. Higher density creates a more festive, crowded effect
- Size Control - Choose from three size presets:
- Small - Subtle, delicate confetti
- Medium - Balanced, standard size (default)
- Large - Bold, eye-catching pieces

🎯 Smart Target Selection

- Select any frame or element on your canvas to add confetti directly inside it
- If no target is selected, Confetti Genie automatically creates a new frame (500x500px) and generates confetti within it
- Works with any element that can contain children (frames, groups, components, etc.)

🌓 Theme Support

- Automatic Theme Detection - The plugin automatically adapts to your Figma app's theme
- Manual Theme Toggle - Switch between light and dark modes using the theme toggle button for your preferred viewing experience

✨ Visual Details

- Varying Opacity - Each confetti piece has randomized opacity (35%-100%) to create natural depth and visual interest
- Random Rotation - Shapes are randomly rotated for a more organic, scattered appearance
- Random Positioning - Confetti pieces are distributed randomly within your selected target area

How to Use

In Figma Design

1. Launch the Plugin - Run Confetti Genie from the plugins menu
2. Select Target (Optional) - Click on a frame or element where you want confetti. If nothing is selected, a new frame will be created automatically
3. Choose Shapes - Open the Shapes dropdown and select which confetti shapes you want (all are selected by default)
4. Set Density - Adjust the density slider to control how many confetti pieces are generated
5. Choose Size - Select Small, Medium, or Large from the size options
6. Customize Colors (Optional) -

- Add colors to your palette using the color picker, or
- Leave it as is to use random colors

7. Generate - Click either:

- "Generate with Selected Colors" - Uses your custom color palette
- "Generate with Random Colors" - Uses vibrant random colors

In FigJam

1. Run the Plugin - Launch Confetti Genie from the plugins menu
2. Select a Frame or Sticky - Click on any frame or sticky note where you want confetti
3. Configure Settings - Adjust shapes, density, size, and colors as desired
4. Generate - Click your preferred generate button to add confetti to your selection

In Figma Slides

1. Open Plugin - Run Confetti Genie from the plugins menu
2. Select Slide Frame - Click on the slide frame where you want confetti
3. Customize - Set your preferred shapes, density, size, and color options
4. Generate - Create confetti to enhance your slide backgrounds or specific elements

In Figma Buzz

1. Launch Plugin - Run Confetti Genie from the plugins menu
2. Target Selection - Select the frame or element in your Buzz project
3. Adjust Settings - Configure all your preferences for shapes, density, size, and colors
4. Generate - Add confetti to create engaging interactive elements

Tips & Best Practices

- Density Guidelines: Use lower density (30-80) for subtle accents, medium (80-150) for balanced designs, and high (150-300) for maximum celebration impact
- Color Harmony: For brand consistency, add your brand colors to the palette. For fun, spontaneous designs, use random colors
- Size Considerations: Larger sizes work great for backgrounds and hero sections, while smaller sizes are perfect for subtle decorative elements
- Shape Combinations: Mix different shapes for variety, or use a single shape type for a more uniform look
- Multiple Targets: You can select multiple frames at once - confetti will be generated in all selected targets
